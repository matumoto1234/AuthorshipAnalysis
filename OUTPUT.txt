# Description of output

ラベル予測モデルを作成し、それについてホールドアウト検証を行った

具体的には以下の手順で行った

1. データの前処理
いらないラベルを削除
データの型を統一（今回ならObject型）
テキスト部分の処理として、いらない空白、改行を消す、ピリオドなどの点を消す、大文字を小文字に変換を行った
ここがclean_dataラベル

必要なラベルだけをまとめて新たなラベルを作成 labels(name,topic,sign)

2.テキストデータをX軸、ラベルをY軸と置いて、x軸のデータをベクトル化 
テキストデータの単語数をカウント

3.モデルの生成と対応
ロジスティク回帰モデルで重みを計算する
その重みを利用して、多クラス分類モデルでそのラベルかそのラベルじゃないかの2クラス分類をする

生成したモデルに、テキストデータ突っ込んで、それに対応する著者を予測

sklearnのスコアを表示する関数で精度を表示

## output

実際に得られた出力は以下である

```

F1 score:  0.638717446936625
Average precision score:  0.45507633075577
Average recall score:  0.530375
```
